WebChat AutoPost Examples


  • AutoPost Example - Full Page 
    <a href="<autopost.html>"
						Auto-Post WebChat Full Page<a>

    Note: Use this if you are unable to open a new window. It will behave exactly as you would expect an "a" link to work. It will replace the contents of your current browser window with the chat window.


  • AutoPost Example - Popup 
    <a href="#" onclick="window.open("autopost.html",
          'webchat', 'width=400,height=500');">
          AutoPost Popup<a>

    Note: Use this option if you are able to open a new window from your website and specify the size. This is the preferred option as the user will see a new char window overlayed on top of your webpage. Your page MUST be the one that opens and sizes the new window. This is a basic restriction of how browsers work. The webchat window cannot size your new window for you since it did not create that window. That security is baked into every browser.
